输出在html属性里的情况

  • css中允许使用转义字符，\ +ascii 16进制的方法，所以，如果输出点在css中，那么我们可以用这种方法组成payload

  • 常规来说，因为 onxxxx="[输出]" 和 href="javascript:[输出]" 与 <script>[输出]</script> 没有太大区别。因为[输出]所在的地方，都是 javascript 脚本。

    在 HTML 属性中，会自动对实体字符进行转义。

    onxxxx="[输出]" 和 href="javascript:[输出]"中，对实体字符转义完，这两个还能触发，而script在转义的时候触发，所以触发不了了，被当作文本了，而前面两个中的输出，本就是文本形态
